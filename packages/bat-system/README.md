# @funeste38/bat-system

BAT-SYSTEM: sonar (Bat) + ears (BatEars) + wings (BatWings) + fangs (BatFangs) + inversion (BatInversion).

Concept: adaptive request orchestration inspired by bat echolocation.

Status: legacy compatibility layer.

For new code, prefer `@funeste38/bat`, which is the maintained public package and contains the newer canonical implementations plus compatibility aliases.

See `src/demo.ts` for a small local demonstration.
